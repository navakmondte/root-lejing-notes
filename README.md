# root-lejing-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或网站，主要以笔记、工具页面或静态演示为主。

## 项目简介

这是一个个人归档性质的仓库，存放一些零散但可独立运行的 HTML 文件。每个页面通常包含完整的前端代码（HTML + CSS + JavaScript），适合直接通过浏览器打开或托管在静态页面服务上使用。

仓库中内容可能涉及技术笔记、实用小工具、学习示例等，不构成商业产品，也不保证持续更新。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── page-xxx.html       # 独立页面，文件名通常反映内容主题
├── assets/             # 公共资源文件夹（如有）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

每个 HTML 页面均为独立文件，不依赖本仓库外的资源（除非在页面内明确引用 CDN 或外部库）。建议直接克隆或下载后本地浏览。

## 页面归档说明

- 页面内容以个人学习、记录或实验为目的。
- 不保证所有页面均处于最新状态，部分页面可能为早期版本。
- 页面中不包含任何形式的跟踪、广告或用户数据收集代码。
- 如有引用外部资源，会在页面源码中注明来源。

## 维护说明

本仓库不定期更新，新增或修改页面时会提交记录。欢迎通过 Issues 提出建议或报告问题，但不保证即时响应。Fork 或自由使用仓库中的代码时，请遵守仓库的许可证要求。
